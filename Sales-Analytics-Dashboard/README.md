# Sales Analytics Dashboard (Power BI)

## Overview
This Power BI dashboard analyzes sales performance across regions, products, and time to help stakeholders track revenue trends and key business KPIs.

## Key KPIs
- Total Revenue
- Sales Growth (MoM / YoY)
- Top Products and Regions
- Customer Contribution

## What I Built
- Star schema data model for analytics
- DAX measures for KPIs, trends, and comparisons
- Interactive slicers and drill-through pages
- Performance-optimized visuals

## Tools & Skills
- Power BI Desktop
- DAX
- Power Query
- Data Modeling
- SQL (data source preparation)

## Files
- Power BI Template: `sales_dashboard.pbix`
- PDF Export: `dashboard.pdf`


## How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
